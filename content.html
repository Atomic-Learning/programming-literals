A literal is a fixed value written directly in your code. Literals represent themselves—they are the actual data, not a reference to data stored elsewhere.

Literals vs Variables

Variables are names that refer to values. Literals are the values themselves.

age ← 25

# "age" is a variable (a name)
# 25 is a literal (the actual value)

Types of Literals

Different data types have different literal formats:

  • Numbers: 42, 3.14, -100
  • Text (strings): "hello", "world"
  • Boolean: true, false

Using Literals

You can use literals directly in expressions:

result ← 10 + 5
# 10 and 5 are numeric literals

message ← "Hello" + " " + "world"
# "Hello", " ", and "world" are string literals

Or assign them to variables:

name ← "Alice"
# "Alice" is a string literal
# name is a variable that now refers to that value
